## How to start the project:

1. Clone this repository
2. Install dependencies
3. Create an auth0 account [https://auth0.com/]
4. Create a new SPA in auth0
5. Fill the environment variables
6. Run the project

### commands

```bash
    # project
    pnpm dev
    pnpm build
    pnpm lint
    pnpm test:cov
    pnpm test:ui
```
